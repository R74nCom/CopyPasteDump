How It Works

Use this text tool to quickly convert English to Minecraft Enchanting Table language, also known as Standard Galactic Alphabet (SGA). Simply type or paste text into the first box, and enchantment table language will appear in the second box for you to copy.

This translator works by converting each letter into the most similar Unicode character in the Standard Galactic Alphabet.

The Standard Galactic Alphabet is a fictional alien language (a conlang) that is present in multiple games, like Minecraft and Commander Keen.

This can be used to translate the text that appears in Minecraft, but the enchanting table uses random English words, and they don't have anything to do with the enchantment. So, no cheating there!
