How It Works

Use this text tool to quickly convert text to Wingdings, which are what W. D. Gaster from Undertale uses. Simply type or paste text into the first box, and Wingdings will appear in the second box for you to copy.

Wingdings is a "dingbat" font developed by Microsoft for Windows in 1990. They made each character on the keyboard look like a symbol for decorating documents with.

W. D. Gaster uses this in-game when talking, but only uses the uppercase variants of symbols.

You can also use the dedicated Text to Wingdings Converter.
