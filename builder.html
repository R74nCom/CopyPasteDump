Emoji Builder

This tool can help you create custom emoji families, couples with hearts, holding hands, and other Zero Width Joiner sequences.



Family Builder

On all platforms except Windows, the order of family members matter.
For parents, man goes before woman. For children, girl goes before boy.
Only Windows fully supports baby children and mixing colorless and colored.




Holding Hands Builder

On all platforms, Woman must go before Man where applicable.
Only Windows supports mixing colorless and colored people in the same emoji.
No platforms currently support mixing the genderless Adult with Man or Woman.


Couple with Heart Builder

On all platforms except Windows, the order of family members matter:
Woman goes before Man where applicable.
Only Windows fully supports skin tones for these.


Kiss Builder

On all platforms except Windows, the order of family members matter:
Woman goes before Man where applicable.
Only Windows fully supports skin tones for these.
