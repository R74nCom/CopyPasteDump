Non-RGI Emoji List

This is a list of emojis used by different platforms that aren't in the Unicode Standard. These emojis are called non-RGI. RGI stands for Recommended for General Interchange, which is a designation in the Unicode Standard that recommends an emoji be rendered on all platforms.

Most of these can also be found on the platform's respective page by clicking on its name.

Many are ZWJ (Zero Width Joiner) sequences, which means you will see the emoji components if you are not using the platform.
