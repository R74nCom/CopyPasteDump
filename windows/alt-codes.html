Windows Alt Codes

Back to the Windows Portal

Alt codes are keyboard shortcuts that let you easily type a special text symbol. Alt codes can be done by holding down the Alt key on the numpad, and pressing each number in the code. If you don't have a numpad, try holding Alt and Fn at the same time. If Numlock is disabled, this may not work right.
